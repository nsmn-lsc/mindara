Características del Sistema

3 Niveles de Usuario

Sistema jerárquico con permisos granulares: ADMIN, MANAGER y USER

Autenticación JWT

Tokens seguros con renovación automática. Sesiones de 60 minutos con refresh de 7 días.

API REST Completa

Endpoints para registro, login, gestión de usuarios y perfiles. Lista para integrar con React.

Endpoints Principales de la API

POST /api/auth/login/ - Iniciar sesión y obtener tokens JWT
POST /api/auth/register/ - Registrar nuevo usuario
GET /api/auth/profile/ - Obtener perfil del usuario autenticado
GET /api/auth/users/ - Listar usuarios (según permisos)
GET /api/auth/stats/ - Estadísticas del sistema (solo admins)
